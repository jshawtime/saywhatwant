# Videos to Upload

Place videos here to sync to Cloudflare R2.

## Naming Convention

- Background videos: `sww-XXXXX.mp4`
- Entity intros: `[entity-id].mov` (e.g., `the-eternal.mov`)

## Sync Command

```bash
./scripts/sync-videos-to-r2.sh
```

See `docs/221-VIDEO-SYSTEM-AND-ENTITY-INTROS.md` for full documentation.
